Computer Vision • University Project

Ippo Fighting Game

Developed as a duo university project at ISEL, the goal was to create an interactive experience using Computer Vision. By implementing image processing techniques like binarization and adaptive thresholding, we developed a system that translates real-time hand gestures and movement, into game commands.

Tools like Unity and Jupyter Notebook were essential in making this project's real-time processing possible.

Step into the shoes of Ippo, using your own movements to control the boxer’s strikes and defeat the opponent in this immersive technical challenge.

For a detailed breakdown of the development process and algorithms used, visit the page.
